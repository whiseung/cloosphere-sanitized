## 왜 Cloosphere인가요?

### 🔒 기업 수준의 보안
- **사설 네트워크 환경**: 모든 데이터가 기업 내부 네트워크에서만 처리됩니다
- **역할 기반 접근 제어**: 조직 구조에 맞는 세밀한 권한 관리
- **완전한 감사 로그**: 모든 사용자 활동을 투명하게 추적

### 🚀 생산성 극대화
- **맞춤형 AI 에이전트**: 부서별, 업무별 특화된 AI 어시스턴트 구성
- **지식 기반 연동**: 사내 문서와 데이터베이스를 AI에 즉시 연결
- **도구 통합**: 기존 업무 시스템과 원클릭 연동

### 💰 비용 효율성
- **사용량 추적**: 부서별, 사용자별 비용 모니터링
- **멀티 모델 지원**: 업무에 맞는 최적의 모델 선택
- **온프레미스 옵션**: 클라우드 비용 없이 자체 서버에서 운영

## 빠른 시작

### 1단계: 로그인
회사 계정(Microsoft Entra ID)으로 간편하게 로그인하세요.

### 2단계: 모델 선택
사용 목적에 맞는 AI 모델을 선택하세요.

### 3단계: 대화 시작
질문을 입력하고 AI와 대화를 시작하세요.

➡️ [상세 가이드 보기](./getting-started.md)
