## 작업 모델 (Task Model)

제목 생성, 태그 생성, 후속 질문 추천 등 **백그라운드 자동화 기능**에 사용되는 모델을 지정합니다.

| 설정 항목 | 설명 |
|-----------|------|
| **Local Models** | 시스템에 등록된 모델 중에서 선택 |
| **External Models** | 작업 모델로 별도 외부 API 사용 시 모델 ID 입력 |

<Tip>
  작업 모델은 빠른 응답 속도와 저렴한 비용이 핵심입니다. 경량 모델(예: GPT-4o-mini, Claude Haiku, Gemini Flash)을 권장합니다.
</Tip>

<Warning>
  작업 모델이 미설정이면 **현재 채팅에 선택된 모델**이 백그라운드 작업에도 사용됩니다.
  고가 모델(Opus, GPT-4o 등)이 선택된 채팅에서는 매 턴마다 제목/태그/후속 질문 생성에도 같은 모델이 호출되어 비용이 크게 증가할 수 있습니다. 운영 환경에서는 반드시 작업 모델을 별도 지정하세요.
</Warning>

## 자동 생성 기능

채팅 관련 자동 생성 Task의 활성화 여부와 프롬프트를 설정합니다. 각 Task는 **토글로 ON/OFF**, **프롬프트 템플릿 커스터마이즈**가 모두 가능합니다.

| Task | 동작 | 기본값 |
|------|------|:------:|
| **Title Generation** (제목 생성) | 첫 턴 후 채팅 대화 내용 기반 자동 제목 생성 | ON |
| **Tags Generation** (태그 생성) | 채팅 내용에서 자동으로 태그 추출·부여 | ON |
| **Follow-Up Generation** (후속 질문 추천) | 마지막 어시스턴트 응답 아래에 3~5개의 후속 질문을 버튼 목록으로 제안 | **OFF** |
| **Autocomplete Generation** (자동완성) | 사용자 입력 중 메시지 자동완성 제안 | ON |

<Tip>
  **Follow-Up Generation은 기본 OFF**로 출시되었습니다. 사용자에게 후속 질문 버튼을 보여주려면 매 턴 추가 LLM 호출이 발생하므로, 비용·응답 지연을 고려해 켜세요.
  켜면 어시스턴트 응답이 끝난 직후 응답 하단에 세로 버튼 목록으로 추천 질문이 노출됩니다.
</Tip>

### 프롬프트 커스터마이징

각 Task의 프롬프트를 비워두면 시스템 기본 프롬프트가 사용되고, 입력하면 그 프롬프트가 우선 적용됩니다.

| 프롬프트 | 설명 |
|----------|------|
| **Title Generation Prompt** | 채팅 제목을 어떤 길이/스타일로 생성할지 |
| **Tags Generation Prompt** | 어떤 카테고리/언어로 태그를 추출할지 |
| **Follow-Up Generation Prompt** | 후속 질문의 톤·관점·개수(예: "사용자의 다음 행동을 예측하는 3개의 짧은 질문") |

### Autocomplete Input Max Length

자동완성을 트리거할 입력 길이의 상한을 지정합니다.

| 값 | 의미 |
|:---|:---|
| `-1` | 길이 제한 없음 (모든 입력에서 자동완성 시도) |
| `1` 이상 양의 정수 | 해당 글자 수 이하일 때만 자동완성 호출 |

긴 메시지에서 매번 자동완성을 호출하면 비용·지연이 누적되므로, 운영 환경에서는 적절한 상한(예: `200`)을 지정하길 권장합니다.

## 프롬프트 제안

새 채팅 시작 시 사용자에게 표시되는 기본 프롬프트 제안을 설정합니다.

각 제안은 **제목(Title)**, **부제목(Subtitle)**, **내용**(실제 전송될 프롬프트)으로 구성됩니다. **"+" 버튼**으로 추가하고, 각 항목의 **"×" 버튼**으로 삭제합니다.

<Tip>
  에이전트별로 다른 프롬프트 제안이 필요하면, 에이전트 편집 화면의 **프롬프트 제안 설정**에서 Custom으로 개별 설정하세요. 에이전트별 설정이 이 전역 설정보다 우선합니다.
</Tip>

## 배너

시스템 전체에 공지 배너를 표시합니다. 유지보수 예정, 정책 변경 등 중요 공지에 활용합니다.

| 설정 항목 | 설명 |
|-----------|------|
| **배너 유형** | `info`, `warning`, `error`, `success` |
| **내용** | 배너에 표시할 메시지 (HTML 지원) |
| **제외가능** | 사용자가 배너를 닫을 수 있는지 여부 |
