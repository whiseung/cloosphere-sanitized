워크스페이스의 에이전트, 지식기반, 플로우, 가드레일, 도구, 프롬프트, 데이터베이스, 용어집 — 리소스가 많아지면 원하는 항목을 찾기 어려워집니다.

태그는 **모든 워크스페이스 리소스에 공통으로 붙일 수 있는 라벨**입니다. "마케팅", "고객지원", "실험중" 같은 태그를 만들어 리소스를 분류하면, 각 목록에서 태그로 필터링할 수 있습니다.

---

## 태그 관리

**워크스페이스 > Tags** 탭에서 태그를 생성·수정·삭제합니다.

### 태그 생성

상단의 입력 필드에 태그 이름을 입력하고 **Create** 버튼을 클릭합니다. 동일한 이름의 태그는 중복 생성할 수 없습니다.

### 태그 수정

태그 행의 **편집 아이콘**을 클릭하면 이름을 인라인으로 수정할 수 있습니다. Enter로 저장, Esc로 취소합니다.

### 태그 삭제

태그를 삭제하면 **모든 워크스페이스 리소스에서 해당 태그가 제거**됩니다.

<Warning>
  태그 삭제는 되돌릴 수 없습니다. 삭제 전 확인 다이얼로그가 표시됩니다.
</Warning>

---

## 태그 적용

각 워크스페이스 리소스(에이전트, 지식기반 등)의 편집 화면에서 태그를 붙이거나 뗄 수 있습니다.

**지원 리소스 유형:**

| 리소스 | 태그 적용 위치 |
|--------|--------------|
| **에이전트** | 에이전트 편집 화면 |
| **플로우** | 플로우 편집기 |
| **지식기반** | 지식기반 편집 화면 |
| **데이터베이스** | DbSphere 편집 화면 |
| **가드레일** | 가드레일 편집 화면 |
| **프롬프트** | 프롬프트 편집 화면 |
| **도구** | 도구 편집 화면 |
| **용어집** | 용어집 편집 화면 |

## 권한

| 작업 | 필요 권한 |
|------|----------|
| 태그 조회 | 로그인한 모든 사용자 |
| 태그 생성 | 로그인한 모든 사용자 |
| 태그 수정/삭제 | 태그 생성자 또는 Admin |
| 리소스에 태그 부착/해제 | 로그인한 모든 사용자 |
