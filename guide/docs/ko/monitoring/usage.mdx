사용량 대시보드는 AI 리소스 소비 현황을 **실시간으로 시각화**합니다.
토큰 사용량, 요청 수, 비용을 다양한 차원(모델, 사용자, 에이전트, 그룹, 팀)으로 분석할 수 있습니다.

**관리자 > 모니터링 > 사용량** 탭에서 접근합니다.

## 주요 지표

대시보드 상단에 핵심 지표가 카드 형태로 표시됩니다.

| 지표 | 설명 |
|------|------|
| **총 토큰** | 선택 기간 내 사용된 전체 토큰 수 |
| **총 요청** | API 호출 횟수 |
| **고유 사용자** | 기간 내 활동한 사용자 수 |
| **고유 채팅** | 생성된 채팅 세션 수 |
| **고유 모델** | 사용된 모델 종류 수 |
| **평균 토큰/요청** | 요청당 평균 토큰 소비량 |

<Tip>
  평균 토큰/요청이 비정상적으로 높다면, 과도한 system prompt나 불필요한 context가 포함되어 있을 수 있습니다. 에이전트 설정을 점검하세요.
</Tip>

## 사용량 추이 차트

시간별 사용량 변화를 라인 차트로 시각화합니다.

<Frame caption="사용량 추이 차트 — 날짜별 토큰 라인 그래프">
  <img src="/images/monitoring/usage-trend-chart.png" alt="사용량 추이 차트 — 날짜별 토큰 라인 그래프" />
</Frame>

### 기간 선택

| 기간 | 집계 단위 |
|------|----------|
| **최근 7일** | 일(day) 단위 |
| **최근 30일** | 일(day) 단위 |
| **사용자 지정** | 날짜 범위 직접 선택 |

### 차트 지표

차트에는 **토큰 사용량 추이**가 라인 그래프로 표시됩니다.

| 지표 | 설명 |
|------|------|
| **토큰** | 날짜별 총 토큰 사용량 |

## 다차원 분석

<Tabs>
  <Tab title="모델별">
    어떤 AI 모델이 얼마나 사용되는지 확인합니다.

    ![모델별 사용량 테이블](/images/monitoring/usage-by-model.png)

    | 컬럼 | 설명 |
    |------|------|
    | **모델 ID** | 사용된 모델 식별자 |
    | **총 토큰** | 해당 모델의 토큰 소비량 |
    | **요청 수** | 해당 모델의 API 호출 횟수 |
  </Tab>
  <Tab title="사용자별">
    개별 사용자의 사용량을 확인합니다.

    | 컬럼 | 설명 |
    |------|------|
    | **사용자** | 이름, 이메일 |
    | **총 토큰** | 사용 토큰 수 |
    | **요청 수** | 요청 횟수 |
  </Tab>
  <Tab title="에이전트별">
    워크스페이스 에이전트별 사용 현황입니다.

    | 컬럼 | 설명 |
    |------|------|
    | **에이전트** | 에이전트 이름 |
    | **총 토큰** | 해당 에이전트의 토큰 소비량 |
    | **요청 수** | 해당 에이전트의 요청 횟수 |

    <Tip>
      에이전트별 비교를 통해 비효율적인 에이전트를 식별하고 프롬프트를 최적화하세요.
    </Tip>
  </Tab>
  <Tab title="그룹별">
    권한 그룹별 사용량을 집계합니다.

    | 컬럼 | 설명 |
    |------|------|
    | **그룹** | 그룹 이름 |
    | **총 토큰** | 그룹 소속 사용자의 총 토큰 |
    | **요청 수** | 그룹 소속 사용자의 총 요청 |
    | **사용자 수** | 그룹 내 활동 사용자 수 |
  </Tab>
  <Tab title="팀별">
    팀(Teams) 단위별 사용량을 확인합니다.

    | 컬럼 | 설명 |
    |------|------|
    | **팀** | 팀 이름 |
    | **총 토큰** | 팀 소속 사용자의 총 토큰 |
    | **요청 수** | 팀 소속 사용자의 총 요청 |
    | **사용자 수** | 팀 내 활동 사용자 수 |
  </Tab>
</Tabs>

## 사용 유형별 분류

토큰이 어떤 작업에 사용되었는지 분류합니다.

<Frame caption="사용 유형별 분포">
  <img src="/images/monitoring/usage-bottom.png" alt="사용 유형별 분포" />
</Frame>

| 유형 | 설명 |
|------|------|
| **CHAT** | 일반 채팅 대화 |
| **EMBEDDING** | 문서 임베딩 처리 |
| **TITLE_GENERATION** | 채팅 제목 자동 생성 |
| **TAGS_GENERATION** | 채팅 태그 자동 생성 |
| **QUERY_GENERATION** | RAG 검색 쿼리 생성 |
| **TOOL_CALL** | 도구 호출 |
| **IMAGE_GENERATION** | 이미지 생성 |
| **IMAGE_PROMPT_GENERATION** | 이미지 프롬프트 생성 |

## 필터링

다양한 조건을 조합하여 데이터를 필터링할 수 있습니다. 모든 필터는 동시 적용됩니다.

| 필터 | 설명 |
|------|------|
| **기간** | 시작/종료 날짜 범위 |
| **모델** | 특정 모델 선택 |
| **사용자** | 특정 사용자 선택 |
| **그룹** | 특정 그룹 선택 |
| **팀** | 특정 팀 선택 |
| **에이전트** | 특정 에이전트 선택 |

<Note>
  필터 드롭다운에는 실제 사용 기록이 있는 항목만 표시됩니다.
</Note>

<Note>
  개별 대화의 상세 내역은 별도의 **모니터링 > 대화 로그** 탭에서 확인할 수 있습니다.
</Note>

## 데이터 내보내기

사용량 데이터를 **CSV** 또는 **JSON**으로 내보낼 수 있습니다. 내보내기 버튼은 대시보드 우측 상단에 위치합니다.

<Tip>
  월별 CSV를 다운로드하여 비용 보고서를 작성하거나, 부서별 사용량을 스프레드시트에서 분석하세요.
</Tip>
