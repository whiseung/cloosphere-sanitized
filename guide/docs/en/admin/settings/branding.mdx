---

In **Admin > Settings > Branding**, customize logos, icons, and loading screens to match your corporate identity.

<Note>
  **License required:** Available on Professional tier and above. On Basic/Standard tiers, the default Cloosphere branding applies.
</Note>

## Customizable Items

### Logo

| Item | Description | Where Shown |
|------|-------------|-------------|
| **Logo (Light)** | Light mode logo (PNG) | Sidebar, sign-in screen |
| **Logo (Dark)** | Dark mode logo (PNG) | Sidebar, sign-in screen (dark mode) |

### Splash Image

| Item | Description | Where Shown |
|------|-------------|-------------|
| **Splash (Light)** | Light mode loading screen image (PNG) | App initial loading screen |
| **Splash (Dark)** | Dark mode loading screen image (PNG) | App initial loading screen (dark mode) |

### Others

| Item | Description | Recommended |
|------|-------------|-------------|
| **App Icon** | PWA install icon | 512x512 PNG |
| **Favicon** | Browser tab icon | ICO or PNG |

---

## App Name

| Setting | Description |
|---------|-------------|
| **App Name** | Name shown across the interface — browser tab title, sidebar, sign-in screen, etc. |

---

## How to Upload Images

<Steps>
  <Step title="Open the Branding tab">
    Go to **Admin > Settings > Branding**.
  </Step>
  <Step title="Upload images">
    Click each item's upload area or drag and drop an image file.
  </Step>
  <Step title="Save and verify">
    Branding assets like logo and favicon save automatically on upload. **App Name** requires clicking save after editing.
  </Step>
</Steps>

---

## Reset

Click the **Reset button** to the right of each item to restore the default Cloosphere image.

<Note>
  Branding changes apply immediately to all users. Some users may see delayed updates due to browser cache.
</Note>
