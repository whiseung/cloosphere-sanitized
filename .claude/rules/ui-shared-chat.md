# 공유 채팅 뷰어 규칙

## 용도
- 공개 공유 채팅 읽기 전용 뷰어
- 비로그인 사용자도 접근 가능

## 라우트 구조
```
src/routes/s/[id]/+page.svelte
```
- 동적 `[id]` 파라미터로 공유 채팅 로드
- `(app)` 그룹 외부 — 별도 인증 모델

## 핵심 기능
- 공유 채팅 데이터 로드 (공개 API)
- 채팅 메시지 컴포넌트 재사용 (읽기 전용)
- 공유 채팅 복제 기능 (로그인 사용자)

## 참조 파일
- `src/routes/s/[id]/+page.svelte`: 공유 채팅 뷰어 페이지
